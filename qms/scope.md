# QMS Scope

## 1. Purpose
This document defines the boundaries and applicability of the Quality Management System (QMS) for the **paser-mini** project, ensuring that all development and operational activities align with the ISO 9001:2015 standard.

## 2. Scope of Application
The QMS applies to the entire lifecycle of the paser-mini autonomous agent, including:
- **Software Development**: Design, implementation, and testing of the ReAct core engine and tool ecosystem.
- **Infrastructure Management**: Integration with LLM providers (Gemini, NVIDIA, etc.) and persistent memory systems.
- **Quality Assurance**: Validation of tool execution, error handling, and non-conformity management.
- **Documentation**: Maintenance of technical specifications, user guides, and QMS records.

## 3. Products and Services
The primary product is the **paser-mini autonomous agent**, a CLI-based tool capable of executing complex tasks via a modular toolset and LLM orchestration.

## 4. Justification for Non-Applicability
All clauses of ISO 9001:2015 are considered applicable to this project. There are no exclusions.
