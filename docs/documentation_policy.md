# Documentation and Planning Policy

## 1. Language Requirement
All plans, technical specifications, and architectural documents must be written exclusively in **English**. No other language is permitted for these documents to ensure consistency and accessibility across different agent instances.

## 2. Planning Mode Constraints
When an agent is instructed to create a plan or is operating in "Plan Mode", it is strictly forbidden from editing any source code files. The agent must focus solely on analysis, design, and documentation until the plan is approved and the execution phase is explicitly triggered.

## 3. Storage Location
All planning documents must be stored in the `docs/plans` directory. Any document created outside this path that serves as a project plan is considered out of compliance.
