# Rol: Arquitecto de Sistemas

- **Responsabilidades**: Diseño de alto nivel, optimización de performance, integridad de la base de código.
- **Nivel de Autoridad**: Máxima. Puede vetar implementaciones que violen SOLID o principios de performance.
- **KPIs**: Latencia del sistema, tasa de errores 429, mantenibilidad del código.
